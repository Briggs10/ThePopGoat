🎤 THE POP G.O.A.T

An interactive browser-based trivia game where players rise through pop culture challenges to earn their place as the POP G.O.A.T 🐐👑. Create your persona, answer questions, unlock rewards, and perform your own pop anthem at the end.

🚀 Live Demo
👉 https://briggs10.github.io/ThePopGoat/

🎮 Features

🎤 Persona Creation:
    Custom player name (Moniker),
    Selectable avatar.
		
🧠 Pop Culture Trivia Levels:
    Multiple-choice questions,
    Progression-based gameplay,
    Immediate feedback on answers.
		
🛍️ Unlockable Rewards System and earn items as you progress:
    Custom Outfit,
    Designer Shoes,
    Certified Jewelry,
    POP G.O.A.T Crown.
		
🎶 Interactive Finale:
    Write your own 30-second pop song,
    Text-to-speech performance,
    Animated avatar stage performance.
		
🎉 Visual Effects:
    Confetti celebration on win,
    Animated UI (hover effects, glowing text, dancing avatar).
		
🔒 Built-in Profanity Filter:
    The built-in profanity filter keeps gameplay clean and appropriate for the interactive finale. 
		
🧩 How to Play:
    Click ENTER on the welcome screen,
    Create your POP G.O.A.T persona, 
    Progress through trivia levels,
    Answer correctly to advance,
    Unlock rewards along the way,
    Reach the final stage:
    Write your pop song,
    Perform it live 🎤,
    Celebrate your G.O.A.T status 🐐👑!
		
🛠️ Tech Stack:
    HTML5,
    CSS3,
    Custom animations,
    Neon/glow effects,
    JavaScript,
    Game logic,
    DOM manipulation,
    Speech Synthesis API.
		
⚙️ Setup & Deployment:
    Run Locally,
    Download or clone the repo.
    Open index.html in your browser.
